
Application details:
1. FindMyService is a SOAP web service
2. FindMyServiceApp is a java web client
3. FindAService is the android project.
3. Postgres with PostGIS is required.
4. postgresServer file from the deliverables should be imported to restore the server


This application was developed using- 
OS: Windows 10 64bit
IDE: Eclipse Oxygen, Netbeans IDE, Android studio
Database: Postgres 10 with PostGIS 
Java 1.8
